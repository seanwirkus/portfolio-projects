Circuit Lab

A Tinkercad-style circuit workbench in the browser: a growing library of passives, power, semiconductors, and meters, with pan/zoom canvas wiring and a real DC modified nodal solver for resistive networks and ideal sources.

Embedded here: the full workbench. Use Wire to connect pins, GND for the reference, and the inspector to edit values. Scroll zooms, middle-click pans.

Modularity One registry, many part types

Parts are defined in a single catalog; adding a new type updates the palette and simulation hooks in one pass.

DC core MNA + net merging

Resistors, ideal voltage sources, and iterative comparator output drive the linear solve with a shared ground net.

Build direction AC & SPICE next

This release targets teaching and quick topology checks; time-domain and small-signal are natural extensions.

Run a local static server for the best resize behavior; the iframe loads the same bundle as the direct runtime.

Tech stack

React SVG canvas MNA (DC) esbuild

What the embed shows

A schematic surface with a searchable part library, bezier-style wires, and readouts for LEDs, voltmeters, and ammeters that reflect the last DC solve.

Scope note

  • Semiconductors use simplified or toggled models suitable for learning, not foundry-accurate SPICE.
  • AC, transient, and autowired breadboard views are not in this build.
