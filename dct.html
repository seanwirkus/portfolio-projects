About the Project

Delivery Center Tycoon is a logistics simulation game with a complete supply chain loop. Players manage inbound shipments, sort and stock inventory across warehouse shelves, fulfill outbound contracts by packing boxes, and dispatch delivery vans. The game includes a companion React/TypeScript web dashboard for analytics and management.

Key Features

  • Full supply chain: inbound air shipments, unloading, sorting, stocking, packing, dispatch
  • Server-authoritative inventory system with real-time client updates
  • Warehouse upgrade progression with expanding SKU variety
  • 3D item models and visual sorting tables with SurfaceGui labels
  • Companion React web dashboard with analytics and charts
  • Python tooling for procedural asset generation

Game Systems

The game implements a server-authoritative inventory model where shelf contents drive the canonical stock state. Outbound contracts are generated from available inventory, requiring minimum unit thresholds before offers appear. The progression system expands from 5 to 20 SKUs as players advance.

Companion Dashboard

A separate React/TypeScript app built with Vite provides warehouse analytics, inventory visualization with Recharts, and management tools. The dashboard uses Radix UI components, Zustand for state management, and a polished UI with Tailwind CSS.
