RetroVision

A retrofit perception HUD for legacy vehicles.

RetroVision brings modern driver-assist awareness to older cars through camera-based object detection, LiDAR distance sensing, and a real-time ESP32-driven display interface. It is designed for driver safety, low-cost ADAS retrofits, embedded perception research, and vehicle prototyping.

ESP32-S3 ESP32-C3 UART LVGL LiDAR
Perception HUD

Real-time objects, distances, and alerts.

Sensor Fusion

Camera + LiDAR over a two-controller UART link.

Orbital

A molecule graph, renderer, and planning surface for chemistry workflows.

Orbital connects a real molecule model to editing tools and visualization — atoms and bonds resolve through the same engine you ship, so sketches in the browser behave like the code behind them.

React Python Chemistry Model-driven UI
Molecule graph

Atoms and bonds stay coherent as you edit — one model drives placement, bonding, and erase tools.

Renderer + tools

The live canvas uses the project’s real renderer classes — not a decorative placeholder.

Workflow surface

Built as a planning and teaching layer on top of chemistry intelligence — extend toward synthesis routes when you need them.
