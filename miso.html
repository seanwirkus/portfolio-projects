miSO

A native macOS virtual machine manager with a guided wizard interface for creating, configuring, and running VMs. Built with Swift using MVVM architecture, miSO provides a clean step-by-step workflow for setting up virtual machine configurations and managing VM instances.

About the Project

miSO simplifies virtual machine management on macOS through a wizard-driven UI. Users step through configuration options to set up VMs, then launch and manage them from a central list view. The app leverages Apple's Virtualization framework for native VM performance.

Key Features

  • Step-by-step VM creation wizard with guided configuration
  • VM list view for managing multiple virtual machines
  • Native macOS Virtualization framework integration
  • MVVM architecture with clean separation of concerns
  • VM display view for interacting with running instances

Architecture

The app follows MVVM with a WizardContainerView orchestrating multi-step VM creation, VMListView for instance management, and VMDisplayView for live VM interaction. The Models layer defines VMConfiguration and WizardStep data, while Services handle Virtualization framework bridging.

Native Performance

By building on Apple's Virtualization framework rather than third-party hypervisors, miSO achieves near-native VM performance on Apple Silicon Macs with minimal overhead and tight system integration.
