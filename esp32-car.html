About the Project

This embedded systems project pairs an AI-Thinker ESP32-CAM with an ultrasonic sensor for obstacle detection, a LiquidCrystal I2C display for real-time feedback, and high-speed 2Mbaud UART communication to an ESP32-S3 display unit. The car supports live camera streaming alongside sensor data.

Key Features

  • Live camera streaming via ESP32-CAM module
  • HC-SR04 ultrasonic distance sensing for obstacle avoidance
  • I2C LCD display for real-time telemetry readout
  • 2Mbaud UART link to ESP32-S3 companion display
  • Wireless receiver module for remote control input

Hardware Architecture

The system is split between an ESP32-CAM handling camera capture and sensor readings, and a separate receiver/display unit. Communication runs over a dedicated UART bus at 2Mbaud for low-latency telemetry and control data transfer between the two boards.

Embedded Stack

Built with PlatformIO targeting the esp32cam board using the Arduino framework. The firmware integrates the ESP camera driver, Wire library for I2C peripherals, and LiquidCrystal_I2C for display output, all running on the ESP-IDF-backed Arduino core.
